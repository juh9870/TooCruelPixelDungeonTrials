# Too Cruel Pixel Dungeon Trials

Too Cruel Pixel Dungeon Trials is a continuation of the Too Cruel Pixel Dungeon, with new modifiers and ported to the modern version of Shattered Pixel Dungeon.

## Installation
Grab an apk or jar from the [releases page](https://github.com/juh9870/TooCruelPixelDungeonTrials/releases/latest).
